In today’s data-driven business environment, procurement is no longer just a transactional function—it is a strategic driver of cost optimization, operational efficiency, and supplier relationship management. Organizations rely heavily on Enterprise Resource Planning (ERP) systems like SAP to manage complex procurement activities within the Procurement-to-Pay (P2P) cycle. Among these, the SAP Materials Management (MM) module plays a crucial role in handling purchasing, vendor management, inventory control, and invoice processing.

This project focuses on performing an in-depth analysis of procurement data using SAP MM datasets to evaluate vendor performance and enhance procurement efficiency. The primary objective is to transform raw transactional data—such as purchase orders, goods receipts, and vendor master records—into meaningful insights that support better decision-making.

A common challenge faced by organizations is the lack of visibility into vendor reliability, delivery timelines, and cost variations across suppliers. Without proper analytics, procurement teams often struggle with inefficient vendor selection, delayed deliveries, and uncontrolled costs, which ultimately impact overall business performance. This project addresses these challenges by applying data analytics techniques to systematically assess vendor efficiency, identify delays in the supply chain, and uncover cost-saving opportunities.

The analysis is built on key SAP tables such as EKKO (Purchase Order Header), EKPO (Purchase Order Items), and LFA1 (Vendor Master), which provide a structured foundation for understanding procurement activities. By calculating and evaluating important Key Performance Indicators (KPIs)—including vendor delivery performance, average procurement cost, delayed delivery percentage, and vendor-wise order distribution—the project provides a comprehensive view of procurement effectiveness.

To extract and visualize these insights, the project utilizes Python within a Jupyter Notebook environment, along with tools like Excel or Power BI for data representation. Various visualizations, such as bar charts, line graphs, and pie charts, are used to clearly communicate trends and patterns, making the analysis intuitive and actionable.

What makes this project unique is its focused approach toward vendor performance analysis within the broader P2P cycle. Instead of covering the entire process superficially, it dives deeper into critical areas like delivery efficiency, cost comparison, and delay tracking—offering practical insights that can directly improve procurement strategies.

Ultimately, this project demonstrates the power of combining SAP ERP data with modern data analytics tools to drive smarter, faster, and more efficient procurement decisions. It highlights how organizations can move from reactive operations to proactive, insight-driven strategies, thereby improving supplier management, reducing costs, and enhancing overall business performance.
